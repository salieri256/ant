# Lake Counting (POJ No.2386)

水たまりの数を数えたい．
8近傍で隣接している水たまりは，繋がっている（1つ）とみなす．
N行，M列のフィールドで，`W`を水たまり，`.`を水たまりでないタイルとする．

## 解説

深さ優先探索（DFS）を使う．
任意の`W`を起点に，そこから繋がっている水たまりを`.`に置き換える．
これを繰り返し，フィールドから`W`がなくなるまでDFSした回数が解となる．
各タイルにつきDFSは0~1回しか呼ばれないので， $O(8 \times N \times M)=O(N \times M)$ で収まる．
