# 三角形

棒を3本選んで，できるだけ周長の長い三角形を作りたい．
その三角形の長さを求める（作れない場合は0とする）．

## 解法

- [O(n^3)時間の解法](./n3.ts)
- [O(nlogn)時間の解法](./nlogn.ts)

`最も長い棒の長さ < 他の2本の棒の長さの和`なら三角形が作れる．

すべての棒の選び方を試すと$O(n^3)$時間で解ける．
棒を長さでソートしておき，連続する3本の選び方を試すと$O(n log n)$時間で解ける．一番長い棒と，次に長い棒2本で三角形にならないなら，その次に長い棒をとってきてもダメなので．
選び方は$O(n)$で収まるが，ソートが最低でも$O(n log n)$かかるので，$O(n + n log n) = O(n log n)$となる．
