## 🤔 なぜ？

### アルゴリズムとはなにか

問題を解決するための手順や計算方法のこと。
アルゴリズムである条件:

- 正しい結果が得られること
- 必ず終わること

### なぜアルゴリズムを学ぶのか

- 問題を効率的に解くため
- バグの少ない安定した動作を実現するため

### よいアルゴリズムとはなにか

- 高速
- 効率的
- 読みやすい
- 多様な場面で使える
