# Ants (POJ No.1852)

長さLの竿の上をn匹のアリが毎秒1のスピードで動く．
アリ同士がぶつかるとそれぞれ反対を向いて動く．
竿の端に到達すると落ちていく．
各アリの位置だけわかり，方向まではわからない．
すべてのアリが竿から落ちるまでにかかる最小時間と最大時間を求めたい．

## 解法

- [O(n)時間の解法](./n.ts)

アリを区別せずに考えると，ぶつかるのではなくすり抜けたように見ることができる．
最小は，一番真ん中に近いアリが近い方の端に到達するまでの時間．
最大は，一番端に近いアリが遠い方の端に到達するまでの時間．
1匹ずつアリの位置を見ていくだけで良いので， $O(n)$ となる．
